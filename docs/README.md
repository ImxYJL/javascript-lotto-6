# 로또

## 기능 목록
- 로또 구입 금액을 입력받는다.
  - 1000으로 나누어떨어지는 금액
  - 로또는 개당 1000원 -> 최소 금액 1000원
  - 이상 금액 예외 처리
- 로또를 발행한다.
  - 숫자 범위: 1 ~ 45
  - 각 숫자는 중복되지 않음
  - 총 6자리의 수
- 발행한 로또를 출력한다.
  - 오름차순으로 정렬해 출력
  - 형식: [번호1, 번호2, 번호3, 번호4, 번호5, 번호 6]
- 당첨 번호를 입력받는다.
- 보너스 번호를 입력받는다.
  - 위의 6자리 수에 더해 보너스 번호 하나를 더 추첨 (위와 중복 x)
- 당첨 통계를 출력한다.
  - 일치하는 수가 3개 이상일 경우부터 당첨금과 그 개수를 출력
  - 총 수익률 출력
    - 소수점 둘째 자리에서 반올림
