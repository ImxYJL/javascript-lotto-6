# 로또 게임

## 기능 목록

- 로또 구입 금액 입력
  - 조건:
  - 예외:
- 로또 발행: 주어진 금액 / 1000 한 만큼 로또를 발행
  - 조건: 1 ~ 45 범위의 겹치지 않는 6개의 수
  - 예외
    - 로또 숫자가 6개가 아닌 경우
    - 로또에 부여된 모든 숫자가 다르지 않은 경우
- 발행된 로또 출력
  - 형식: `[]`로 둘러싸이고 `, `로 구분되는 문자열
    예) `[1, 2, 3, 4, 5]`
- 당첨 번호 입력
  - 형식:
  - 예외
- 보너스 번호 입력
  - 조건
    - 당첨 번호에 이미 포함된 수인 경우
    - 1 ~ 45 사이의 숫자 하나가 아닌 경우
- 발행된 로또들의 결과 확인
  - 규칙: 당첨 번호와 겹치는 숫자가 몇 개인지와 보너스 번호에 따라 등수 결정
    - 1등: 6개 숫자가 모두 일치
    - 2등:
- 당첨 결과 출력
  - 형식: 쓸말
- 수익률 출력
  - 조건: 소수점 둘째 자리에서 반올림
